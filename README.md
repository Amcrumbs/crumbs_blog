# crumbs_blog

## 环境变量

- `ADMIN_PASSWORD`: 单管理员登录密码。
- `ADMIN_SESSION_SECRET`: 管理员 session 签名密钥，生产环境应使用高强度随机字符串。

## 管理模式说明

- 站点采用“单管理员登录 + 公开只读”的轻量后台模式，不是多用户账号系统。
- 管理员登录后可以维护日志、笔记、导航和留言板。
- 私密日志与个人收藏跟随管理员 session 显示。
- 生产环境必须通过 HTTPS 部署，才能正确启用 `Secure` / `__Host-` cookie 策略。
- 该方案适合个人博客的便利级管理与私密保护，不适合存放高敏感机密内容。
