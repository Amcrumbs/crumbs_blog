## Why

当前首页只有欢迎栏，访客只能看到一句问候，无法快速认识站点主人，也无法立即理解这个博客兼工作站的用途。现在补上个人展示区，可以把“我是谁”和“这个站点是做什么的”放在同一阅读路径里，同时为后续在私密区中维护这部分内容建立稳定的数据结构。

## What Changes

- 在首页欢迎栏下方新增公开的个人展示区，使用简洁名片型布局展示身份信息、重点项目、长期方向和兴趣标签。
- 为个人展示区增加双语内容结构，保证中文和英文文案在同一组件结构下稳定呈现。
- 在私密区新增一个小型内容面板，用于查看和编辑首页展示区内容。
- 将首页展示区内容改为独立本地 JSON 文件驱动，供首页公开展示和私密区编辑共用。
- 为重点项目和个人介绍补充更自然的展示文案，减少占位文本和备注式表达。

## Capabilities

### New Capabilities

- `home-profile-showcase`: 覆盖首页欢迎栏下方的公开个人展示区、双语内容呈现和展示布局安全区。
- `profile-showcase-editor`: 覆盖私密区中的个人展示区编辑面板、本地 JSON 内容源和保存规则。

### Modified Capabilities

- None.

## Impact

- 主要影响首页组件、私密区页面、双语字典或展示文案装配逻辑，以及用于读取和保存展示数据的内容模块。
- 预计涉及 `src/app/page.tsx`、`src/components/home-hero.tsx`、`src/app/private/page.tsx`、`src/app/actions.ts`、`src/lib` 下的内容读取/保存模块，以及新增的个人展示 JSON 文件。
- 不新增数据库、不改变现有文章内容目录结构、不改变现有私密解锁机制。
