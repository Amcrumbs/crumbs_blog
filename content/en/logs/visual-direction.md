The visual keywords are dark surfaces, fine borders, compact panels, command-like cues, and a few restrained status colors.

This is not meant to be a pure terminal toy. Notes and logs still need to be comfortable to read, so article surfaces need clear contrast, enough line height, and a controlled reading width.

The interface should feel like a tool in use, not a promotional page.
